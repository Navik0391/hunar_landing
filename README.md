# Hunar Landing

Лендинг для платформы **Hunar** — сервиса поиска проверенных специалистов. Ресурс помогает клиентам находить мастеров по ремонту, репетиторов и автомехаников, а специалистам — искать новые заказы.

Проект построен с использованием современных веб-технологий для обеспечения высокой производительности, плавных анимаций и идеальной адаптивности на любых устройствах.

## 🛠 Технологический стек

- **[Vite](https://vitejs.dev/)** — Современный бандлер и локальный сервер для молниеносной разработки.
- **[Tailwind CSS v4](https://tailwindcss.com/)** — Утилитарный фреймворк для стилизации и создания современного дизайна.
- **HTML5** — Семантическая разметка с использованием компонентного подхода (`<include>` для частей страницы).
- **Vanilla JavaScript (ES6+)** — Взаимодействие с DOM, Intersection Observer для анимаций при скролле, логика 3D-каруселей, мобильного меню и модальных окон.

## 🚀 Начало работы

### Системные требования
Убедитесь, что у вас установлен [Node.js](https://nodejs.org/) (рекомендуется версия 18 или выше).

### 1. Установка зависимостей
Откройте терминал в папке проекта и выполните команду:
```bash
npm install
```

### 2. Запуск сервера для разработки
```bash
npm run dev
```
После запуска проект будет доступен по локальному адресу `http://localhost:5173/`. 
Благодаря флагу `--host`, сервер также доступен в вашей локальной сети (по IP-адресу, например `http://192.168.X.X:5173/`), что очень удобно для тестирования верстки прямо с мобильного телефона или планшета!

### 3. Сборка для продакшена (Production)
```bash
npm run build
```
Команда собирает и минифицирует все ассеты. Итоговые файлы, готовые к загрузке на хостинг, появятся в папке `dist/`.

### 4. Предпросмотр собранной версии
```bash
npm run preview
```
Позволяет локально запустить собранную из папки `dist/` версию проекта, чтобы убедиться, что всё работает корректно перед деплоем.

## ✨ Ключевые особенности верстки

- **Модульная структура:** Код страниц разбит на логические блоки внутри `src/partials/`, что делает проект легко поддерживаемым.
- **Scroll Reveal Анимации:** Элементы плавно появляются при достижении их области видимости (на базе чистого `IntersectionObserver` в `main.js`).
- **Интерактивные 3D-эффекты:** Блок скачивания приложения анимирован с помощью `Coverflow Carousel` (сдвиги, масштабирование).
- **Плавные модальные окна:** Реализовано кастомное модальное окно обратной связи. Защита от сдвига (прыжка) контента при исчезновении скроллбара реализована современным CSS-правилом `scrollbar-gutter: stable`.
- **Snowfall Эффекты:** Фоновая анимация "падающих" иконок инструментов, добавляющая дизайну динамики и живости.
